Formule-overzicht

  • r = cov(X,Y) / (SD(X) · SD(Y))
  • Bivariate b = cov(X,Y) / var(X)
  • Bivariate a = Ȳ - b·X̄
  • Ŷ bivariate = a + bX
  • Ŷ meervoudig = a + b₁X₁ + b₂X₂
  • a meervoudig = Ȳ - b₁·X̄₁ - b₂·X̄₂
  • R² = 1 - SSE/SST
  • Pad totaal effect
    direct effect + som van alle indirecte paden
  • Partiele r
    r₁₂.₃ = (r₁₂ - r₁₃·r₂₃) / √((1-r₁₃²)(1-r₂₃²))
  • Gestand. beta
    β₁ = b₁ · SD(X₁) / SD(Y)
